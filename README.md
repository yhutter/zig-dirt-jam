# Zig Dirt Jam
Rewrite of my Acerola Dirt Jam entry in Zig and Sokol.

## How to run
> :warning: This Project was made with Zig Version `0.15.1`. Other versions are not garanteed to work.

Simply execute the following command:

```bash
cd src
zig build run
```
